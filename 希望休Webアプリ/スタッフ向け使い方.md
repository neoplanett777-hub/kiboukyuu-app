# スタッフ向け使い方

## 使う前に

- 管理者から案内されたURLを開いてください
- スマホでもパソコンでも使えます
- ブラウザは `Edge` `Chrome` `Safari` のどれでも大丈夫です

## ログイン方法

1. ログイン画面を開きます
2. `スタッフ用ログイン` を使います
3. 管理者から渡された `ログイン名` と `パスワード` を入力します
4. `スタッフ画面へ` を押します

## 希望休の入力方法

1. `日付` を選びます
2. 1日ずつ追加する場合は `追加` を押します
3. 複数日ある場合は、下の入力欄へまとめて入れても大丈夫です

入力例:

```text
2026-04-03
2026-04-10
2026-04-18
```

または

```text
2026-04-03, 2026-04-10, 2026-04-18
```

4. `休みの種類` を選びます
   - `公休`
   - `有給`
5. `希望休を送信` を押します

## 注意

- スタッフ画面では、自分の入力だけできます
- 他のスタッフの申請一覧は見えません
- 同じ日付を重複して送ることはできません
- 送信後に内容を直したいときは、管理者へ連絡してください

## よくある質問

### 画面が開かない

- URLが正しいか確認してください
- 社内Wi-Fiにつながっているか確認してください

### ログインできない

- ログイン名とパスワードをもう一度確認してください
- 分からない場合は管理者へ確認してください

### 送信できない

- 日付が入力されているか確認してください
- 休みの種類が選ばれているか確認してください

## 管理者へ伝えるときの例

- 「ログインできません」
- 「送信後に内容を変更したいです」
- 「このURLが開きません」
