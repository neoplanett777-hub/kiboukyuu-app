# 管理者向け配布手順

## スタッフへ案内するもの

- アクセスURL
- 各スタッフの `ログイン名`
- 各スタッフの `パスワード`
- 締切日がある場合はその日時

## スタッフへの案内文例

```text
希望休入力はこちらからお願いします。

URL:
http://127.0.0.1:3210

ログイン名:
（各自のログイン名）

パスワード:
（各自のパスワード）

入力後は「希望休を送信」を押してください。
他のスタッフの申請内容は見えません。
```

## 配布前チェック

1. アプリを起動する
2. ブラウザで画面が開くことを確認する
3. スタッフ用アカウントでログインできるか確認する
4. 管理者画面で一覧確認できるか確認する
5. 必要なら `スタッフ向け使い方.md` をそのまま共有する

## 補足

- 今のURL `127.0.0.1` は、そのパソコンだけで使うURLです
- スタッフの端末から使わせる場合は、社内で見えるURLに変更する必要があります
- その対応が必要なら、次にネットワーク共有できる形へ調整します
